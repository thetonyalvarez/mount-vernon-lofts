# Documents Directory

This directory contains PDF documents for Mount Vernon Lofts website.

## Files

- `mvl-floor-plans.pdf` - Complete floor plan collection (to be added)

## Usage

The PDF files in this directory are:
- Referenced by the floor plans API endpoint for email attachments
- Served as downloadable content for leads
- Protected by requiring form submission before access

## File Requirements

- PDF files should be optimized for web viewing
- Recommended maximum file size: 10MB per PDF
- Files should include appropriate metadata and bookmarks
- Use descriptive filenames that reflect the content

## Security Note

While these files are in the public directory, they are not directly linked from the website. 
Access is intended only through the lead capture process.
